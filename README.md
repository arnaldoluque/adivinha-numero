# Jogo: Adivinha o Número 🎯
Um desafio rápido onde tens de adivinhar o número secreto entre 1 e 10. Mas atenção: tens um limite de jogadas!
🎮 Regras do Jogo

    O computador escolhe um número aleatório.
    Tens apenas 3 tentativas para acertar.
    O programa dá feedback se o teu palpite foi muito alto ou  muito baixo.
    E quando as tentativas esgotam, o jogo termina.

# 🛠️ Tecnologias
    Python

# 💻 Como executar
bash

1. Clona o repositório

2. abra o terminal

3. Entra na pasta
 
4. cd teu-repositorio

5. Executa o jogo
  python adivinha.py
